Example: Simple Earnings Growth Calculation (Python)


def earnings_growth(prev_eps, current_eps):
    """
    Calculate earnings growth percentage.
    """
    return (current_eps - prev_eps) / prev_eps * 100

prev_eps = 12.5  # last quarter EPS
current_eps = 13.8  # current quarter EPS

growth = earnings_growth(prev_eps, current_eps)
print(f"Quarterly EPS growth: {growth:.2f}%")

Financial Formula – Present Value of Cash Flows

We discount projected cash flows using the standard present value formula:

\[ PV = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} \]

Where:

  • \( CF_t \) = cash flow at time \( t \)
  • \( r \) = discount rate
  • \( T \) = total number of periods
